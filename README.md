
# ResumeForge – AI Powered Resume Builder

ResumeForge is an AI-powered resume builder that helps users create professional, ATS-optimized resumes in minutes. The platform uses AI to generate content, improve resume sections, and provide structured templates suitable for modern hiring systems.

---

## Features

- AI-assisted resume generation
- ATS-friendly resume formatting
- Multiple resume templates
- Section-based editing (Education, Experience, Skills, Projects)
- Real-time resume preview
- Export resume as PDF
- Easy customization and editing

---

## Tech Stack

### Frontend
- React
- Tailwind CSS

### Backend
- Node.js
- Express.js

### AI Integration
- OpenAI API / LLM-based text generation

### Database
- MongoDB

---

## How It Works

1. Users enter basic information such as education, skills, and experience.
2. The AI generates professional resume bullet points and summaries.
3. The system formats the data into ATS-friendly resume templates.
4. Users can edit sections and export the final resume as a PDF.
